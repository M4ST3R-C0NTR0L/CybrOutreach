All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release of IceBreaker
- AI-powered email generation with multiple providers (OpenRouter, OpenAI, Anthropic)
- Four email templates: Intro, Follow-up, Break-up, Meeting Request
- Four tones: Professional, Casual, Aggressive, Friendly
- Proven copywriting frameworks: AIDA, PAS, BAB
- Batch processing from CSV files
- Beautiful terminal output with Rich
- Clipboard and file output support
- Rate limiting for batch operations
- Comprehensive test suite
- MIT License
